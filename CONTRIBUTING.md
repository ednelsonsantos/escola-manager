# Contribuindo com o Escola Manager

Obrigado pelo interesse em contribuir! Este projeto é open source sob a licença **GPL-3.0**.

## Como contribuir

### 1. Reportar bugs
Abra uma issue em: https://github.com/ednelsonsantos/escola-manager/issues
Inclua: versão do Node, versão do Windows, passos para reproduzir.

### 2. Sugerir melhorias
Abra uma issue com a tag `enhancement` descrevendo o que você gostaria de ver.

### 3. Enviar código (Pull Request)
```bash
git fork https://github.com/ednelsonsantos/escola-manager
git clone https://github.com/SEU-USUARIO/escola-manager
cd escola-manager
npm install
npm run dev
# Faça suas alterações
git commit -m "feat: descrição da melhoria"
git push
# Abra um Pull Request
```

## Regras de contribuição

- Todo código contribuído é automaticamente licenciado sob GPL-3.0
- Mantenha compatibilidade com Node 18+
- Não adicione dependências com licença proprietária ou incompatível com GPL
- Adicione uma observação no CHANGELOG.md
